Topological Sort (DFS) — LeetCode Problema 0269

📖 Qual e o Problema?

Dado um dicionario alien com palavras ordenadas segundo o alfabeto dessa lingua, descubra a ordem das letras. Se houver contradicao, retorne "".

Este visualizador usa o teste AlienOrderTest_1:
words = ["wrt", "wrf", "er", "ett", "rftt"]
Resultado esperado: "wertf"

Fase 1: comparar palavras adjacentes para extrair arestas de ordenacao: t→f, w→e, r→t, e→r.
Fase 2: DFS topologico sobre as letras para descobrir a ordem: w → e → r → t → f.

🔍 Passos do Algoritmo

1 Registrar letras: para cada palavra, marcar cada letra como "presente" (visited[ch-'a'] = 0)
2 Extrair arestas: comparar palavras adjacentes. Primeira diferenca entre caracteres prev[j] ≠ curr[j] → aresta prev[j] → curr[j]
3 Iniciar DFS: para cada letra presente e ainda nao visitada, chama DFS
4 DFS - marcar em visita: visited[i] = 1. Explorar vizinhos
5 DFS - concluir letra: visited[i] = 2. Inserir na frente do resultado (sb.Insert(0, ch))
6 Resultado: concatenacao das letras em ordem topologica

⏱️ Complexidade de Tempo

O(C) onde C = total de caracteres

BuildGraph itera sobre todos os chars. O DFS roda em O(V+E) onde V ≤ 26 letras e E ≤ 26² arestas.

💾 Complexidade de Espaco

O(1) — tamanho fixo de 26

Tanto a matriz de adjacencia bool[26,26] quanto o array visited[26] tem tamanho constante, independente do input.

💻 Implementacao em C#

// Fase 1: BuildGraph
for (int i = 1; i < words.Length; i++)
{
    for (int j = 0; j < prevWord.Length; j++)
    {
        if (j >= word.Length) return false;     // "abc" > "ab" = invalido
        if (word[j] != prevWord[j])
        {
            adj[prevWord[j]-'a', word[j]-'a'] = true;
            break;                                // so a 1a diferenca importa
        }
    }
}

// Fase 2: DFS topologico
private bool DFS(bool[,] adj, int[] visited, int index, StringBuilder sb)
{
    visited[index] = 1;
    for (int j = 0; j < 26; j++)
        if (adj[index, j])
        {
            if (visited[j] == 1) return false;   // ciclo
            if (visited[j] == 0)
                if (!DFS(adj, visited, j, sb)) return false;
        }
    visited[index] = 2;
    sb.Insert(0, (char)('a' + index));         // inserir na frente
    return true;
}
