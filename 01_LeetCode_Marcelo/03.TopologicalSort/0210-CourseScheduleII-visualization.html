Topological Sort (DFS) — LeetCode Problema 0210

📖 Qual e o Problema?

Dado numCourses cursos (0 a n-1) e uma lista de pre-requisitos [a, b] (para fazer a voce precisa ter feito b), retorne uma ordem valida para completar todos os cursos. Se houver ciclo, retorne array vazio.

Este visualizador usa o teste CanFinishTest_3:
numCourses = 4, prerequisites = [[1,0],[2,0],[3,1],[3,2]]
Resultado esperado: [0, 2, 1, 3]

A solucao usa DFS com 3 estados: nao visitado (0), em visita (1, cinza), concluido (2, preto). Apos o DFS de cada no, ele e adicionado ao resultado (pos-ordem) — apos reverter, temos a ordem topologica.

🔍 Passos do Algoritmo

1 Construir grafo: matriz de adjacencia. Para cada [a,b]: adj[b,a] = true (b e pre-req de a)
2 Iniciar DFS: para cada no ainda nao visitado (visited[i] == 0), chama DFS(i)
3 Marcar em visita: visited[i] = 1 (cinza — no na pilha DFS atual)
4 Visitar vizinhos: para cada vizinho j de adj[i,j]: DFS recursivo se visited[j] == 0
5 Concluir no: visited[i] = 2. Adicionar i ao resultado (pos-ordem → reverter no final)
6 Reverter resultado: pos-ordem revertida = ordem topologica. Retorna array final

⏱️ Complexidade de Tempo

O(V + E)

V = numCourses, E = prerequisites. Cada no e cada aresta sao visitados exatamente uma vez no DFS.

💾 Complexidade de Espaco

O(V²) — matriz adjacencia

A implementacao usa matriz bool[V,V]. Usando lista de adjacencia seria O(V+E). Pilha DFS: O(V) no pior caso.

💻 Implementacao em C#

public int[] CanFinish(int numCourses, int[][] prerequisites)
{
    var adj = new bool[numCourses, numCourses];
    BuildGraph(adj, prerequisites);              // adj[b,a] = true para [a,b]

    var visited = new int[numCourses];           // 0=nao visitado 1=em visita 2=concluido
    var result = new List<int>();
    for (int i = 0; i < numCourses; i++)
        if (visited[i] == 0 && !DFS(adj, visited, i, numCourses, result))
            return new int[] { };               // ciclo!

    result.Reverse();
    return result.ToArray();
}

private bool DFS(bool[,] adj, int[] visited, int i, int n, IList<int> result)
{
    visited[i] = 1;
    for (int j = 0; j < n; j++)
    {
        if (adj[i, j])
        {
            if (visited[j] == 1) return false; // ciclo detectado
            if (visited[j] == 0)
                if (!DFS(adj, visited, j, n, result)) return false;
        }
    }
    visited[i] = 2;
    result.Add(i);                               // pos-ordem
    return true;
}
