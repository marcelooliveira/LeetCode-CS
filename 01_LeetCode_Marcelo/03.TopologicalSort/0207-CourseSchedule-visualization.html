Deteccao de Ciclo com DFS (Grafo Dirigido) — LeetCode Problema 0207

📖 Qual e o Problema?

Dados numCourses e uma lista de prerequisites, determine se e possivel concluir todos os cursos. E possivel se e somente se o grafo de prerequisitos nao tiver ciclo.

Este visualizador usa CanFinishTest_2:
numCourses = 2, prerequisites = [[1,0], [0,1]]
Resultado esperado: false (ha um ciclo: curso 0 precisa do 1, e curso 1 precisa do 0)

Para cada prerequisito [a,b]: "b deve ser feito antes de a", logo a aresta no grafo aponta b → a (adj[b,a] = true). O DFS detecta o ciclo quando encontra um no ja em visita (visited = 1).

🔍 Passos do Algoritmo

1 BuildGraph: Para cada prerequisito [a,b], marcar adj[b,a] = true. A aresta vai de b para a (b deve ser feito antes de a).
2 Loop externo: Para cada no com visited[i] == 0, iniciar DFS a partir dele.
3 DFS - entrar no no: Marcar visited[i] = 1 (grey = "em visita na pilha atual").
4 DFS - explorar vizinhos: Para cada aresta i→j: se visited[j] == 1 → CICLO → retornar false. Se visited[j] == 0 → recursao.
5 Ciclo detectado! Um vizinho ja esta na pilha atual (visited = 1), logo ha dependencia circular. CanFinish retorna false.
6 Resultado final: false — nao e possivel concluir todos os cursos.

⏱️ Complexidade de Tempo

O(V + E)

V = numCourses (nos), E = prerequisites (arestas). O DFS visita cada no e aresta no maximo uma vez.

💾 Complexidade de Espaco

O(V²) — matriz de adjacencia

A matriz bool[V,V] ocupa O(V²). O array visited[V] e a pilha de recursao sao O(V).

💻 Implementacao em C#

public bool CanFinish(int numCourses, int[][] prerequisites)
{
    var adj = new bool[numCourses, numCourses];
    BuildGraph(adj, prerequisites);               // [a,b] → adj[b,a] = true

    var visited = new int[numCourses];
    for (int i = 0; i < numCourses; i++)
        if (visited[i] == 0 && !DFS(adj, visited, i, numCourses))
            return false;
    return true;
}

private bool DFS(bool[,] adj, int[] visited, int i, int numCourses)
{
    visited[i] = 1;                               // grey: in current path
    for (int j = 0; j < numCourses; j++)
    {
        if (adj[i, j])
        {
            if (visited[j] == 1) return false;  // cycle!
            if (visited[j] == 0)
                if (!DFS(adj, visited, j, numCourses)) return false;
        }
    }
    visited[i] = 2;                               // black: fully explored
    return true;
}
