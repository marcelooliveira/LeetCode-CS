🔢 Missing Number

Somatório de Gauss — LeetCode 0268

📖 Problema

Um array de tamanho n contém n inteiros distintos do intervalo [0, n]. Exatamente um número está faltando. Encontre-o.

Teste: MissingNumberTest_1()  →  entrada [3, 0, 1], esperado 2

Ideia: a soma de 0 + 1 + … + n é n(n+1)/2. A diferença entre essa soma ideal e a soma real é o número faltante. O código calcula isso em uma única passagem acumulando (i+1) − nums[i].

🔍 Passos do Algoritmo

1Gauss: soma esperada de [0..n] = n×(n+1)/2. Para n=3: 3×4/2 = 6.
2Loop: a cada iteração soma-se (i+1) − nums[i] ao acumulador result.
3i=0: (0+1) − nums[0] = 1 − 3 = −2. result = −2.
4i=1: (1+1) − nums[1] = 2 − 0 = +2. result = 0.
5i=2: (2+1) − nums[2] = 3 − 1 = +2. result = 2.
6Resultado: result = 2. Número faltante no intervalo [0,3].

⏱ Tempo

O(n) — uma única passagem pelo array.

💾 Espaço

O(1) — apenas a variável result.

💻 C#

public int MissingNumber(int[] nums)
{
    int result = 0;
    for (int i = 0; i < nums.Length; ++i)
        result += (i + 1) - nums[i];
    return result;
}
