📋 Find All Numbers Disappeared

Sign Marking (in-place) — LeetCode 0448

📖 Problema

Array de tamanho n com inteiros em [1, n] — pode haver duplicatas. Retornar todos os números de 1 a n que não aparecem.

Teste: FindDisappearedNumbers_1()  →  entrada [4, 3, 2, 7, 8, 2, 3, 1], esperado [5, 6]

Ideia: Para cada num encontrado, marcar nums[|num|−1] com sinal negativo. Ao final, posições ainda positivas indicam números ausentes.

🔍 Passos do Algoritmo

1Fase 1 — Mark: Para cada i, calcular index = |nums[i]| − 1. Se nums[index] > 0, multiplicar por −1 (marcar como "visto").
2Usar |valor|: Como podemos visitar uma posição já negativa, usar valor absoluto para calcular o índice destino.
3Já marcado: Se nums[index] < 0, a posição já foi visitada — apenas ignorar (número duplicado).
4Fase 2 — Scan: Varrer o array. Posição i com nums[i] > 0 significa que o número i+1 nunca apareceu.
5Ex: Após marcar, nums[4]=8>0 → 5 ausente. nums[5]=2>0 → 6 ausente. Todas as demais posições são negativas.
6Resultado: [5, 6] — os únicos números de 1 a 8 que não aparecem em [4,3,2,7,8,2,3,1].

⏱ Tempo

O(n) — duas passagens lineares: marking + scan.

💾 Espaço

O(1) — in-place, sem estruturas auxiliares (a saída não conta).

💻 C#

public IList<int> FindDisappearedNumbers(int[] nums)
{
    // Fase 1: marcar posições visitadas com sinal negativo
    for (int i = 0; i < nums.Length; i++)
    {
        var index = Math.Abs(nums[i]) - 1;
        if (nums[index] > 0)
            nums[index] *= -1;
    }
    // Fase 2: posições positivas → números ausentes
    var result = new List<int>();
    for (int i = 0; i < nums.Length; i++)
        if (nums[i] > 0)
            result.Add(i + 1);
    return result;
}
