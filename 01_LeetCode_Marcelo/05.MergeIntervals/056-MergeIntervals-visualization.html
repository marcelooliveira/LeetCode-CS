🟩 Merge Intervals

Ordenacao + Varredura Linear — LeetCode 0056

📖 Qual e o Problema?

Dado um array de intervalos, mescle todos os intervalos sobrepostos e retorne um array de intervalos disjuntos.

Saida: [[1,6], [8,10], [15,18]]

Estrategia: Ordenar por inicio de intervalo → iterar e mesclar intervalos que se sobrepoe.

🔍 Passos do Algoritmo

1 Ordenacao: Ordenar intervalos por start (e por end se empatado). Array fica: [[1,3], [2,6], [8,10], [15,18]].
2 Inicializacao: result = [intervalos[0]] = [[1,3]]. Ponteiro para ultimo intervalo mesclado.
3 Verificar Sobreposicao: para cada intervalo i, se result[-1].end >= i.start, eles se sobrepoe.
4 Mesclar: se sobrepoe, atualizar result[-1].end = max(result[-1].end, i.end).
5 Adicionar: se nao sobrepoe, adicionar novo intervalo i a result.
6 Resultado Final: retornar result = [[1,6], [8,10], [15,18]].

⏱️ Complexidade de Tempo

O(n log n)

Ordenacao domina. Varredura linear e O(n).

💾 Complexidade de Espaco

O(n)

Array de resultado pode ter ate n intervalos. Espaco para ordenacao depende do algoritmo.

💻 Implementacao em C#

public int[][] Merge(int[][] intervals)
{
    if (intervals.Length <= 1) return intervals;

    // Ordenar por inicio do intervalo
    Array.Sort(intervals, (a, b) =>
    {
        var comp = a[0].CompareTo(b[0]);
        if (comp == 0)
            comp = a[1].CompareTo(b[1]);
        return comp;
    });

    var result = new List<int[]>();
    result.Add(intervals[0]);
    for (int i = 1; i < intervals.Length; i++)
    {
        // Checar sobreposicao
        if (result.Last()[1] >= intervals[i][0])
        {
            // Mesclar: expandir fim do ultimo intervalo
            if (result.Last()[1] < intervals[i][1])
                result.Last()[1] = intervals[i][1];
        }
        else
            result.Add(intervals[i]);
    }
    return result.ToArray();
}
