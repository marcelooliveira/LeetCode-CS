🟨 Insert Interval

Insercao + Mesclagem Seletiva em 3 Fases — LeetCode 0057

📖 Qual e o Problema?

Dado um conjunto de intervalos nao-sobrepostos ordenados, e um novo intervalo para inserir, retorne um conjunto de intervalos mesclados apos a insercao.

Este visualizador usa InsertTest_MultipleOverlap():
Entrada: intervalos = [[1,2],[3,5],[6,7],[8,10],[12,16]], newInterval = [4,9]
Saida: [[1,2],[3,10],[12,16]]

Estrategia: Tres fases: (1) Adicionar intervalos que terminam antes da insercao; (2) Mesclar intervalos sobrepostos; (3) Adicionar intervalos que comecam apos a insercao.

🔍 Passos do Algoritmo (3 Fases)

1 Fase 1 - Antes: adicionar todos os intervalos que terminam antes de newInterval.start. Exemplo: [1,2].end=2 < newInterval.start=4 → adiciona [1,2].
2 Fase 2 - Overlap: enquanto intervalos sobrepoe com newInterval, mesclar: newInterval.start = min(...), newInterval.end = max(...).
3 Fase 2 - Mesclar: exemplo: newInterval=[4,9] com [3,5] resulta em [3,9]; depois [6,7] resulta em [3,9], etc.
4 Fase 2 - Final: depois de mesclar tudo, adiciona o newInterval final.
5 Fase 3 - Depois: adicionar todos os intervalos que comecam depois de newInterval.end. Exemplo: [12,16].start=12 > newInterval.end=10 → adiciona [12,16].
6 Resultado Final: [[1,2], [3,10], [12,16]]. Insercao com mesclagem concluida.

⏱️ Complexidade de Tempo

O(n)

Varredura linear uma unica vez atraves de todos os intervalos. Sem ordenacao necessaria.

💾 Complexidade de Espaco

O(n)

Array de resultado pode ter ate n intervalos. Sem espaco extra alem do output.

💻 Implementacao em C#

public IList<Interval> Insert(IList<Interval> intervals, Interval newInterval)
{
    var result = new List<Interval>();

    // Fase 1: adicionar intervalos que terminam antes de newInterval
    for (int i = 0; i < intervals.Count; i++)
    {
        if (newInterval.start > intervals[i].end)
            result.Add(intervals[i]);
        else if (newInterval.end < intervals[i].start)
        {
            // Fase 3: add remaining, skip to end
            result.Add(newInterval);
            for (int j = i; j < intervals.Count; j++)
                result.Add(intervals[j]);
            return result;
        }
        else
        {
            // Fase 2: sobreposicao → mesclar
            newInterval.start = Math.Min(intervals[i].start, newInterval.start);
            newInterval.end = Math.Max(intervals[i].end, newInterval.end);
        }
    }
    result.Add(newInterval);
    return result;
}
