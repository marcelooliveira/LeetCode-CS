0347 - Top K Frequent Elements

Dictionary de frequências + OrderByDescending + Take(k) — padrão Top K Elements

Como funciona?

Input de teste: nums = [1, 1, 1, 2, 2, 3], k = 2 → resultado esperado: [1, 2]

Fase 1: Percorre o array e conta a frequência de cada número num Dictionary<int,int>.

Fase 2: Ordena o dicionário por frequência decrescente (OrderByDescending), extrai as chaves e pega os primeiros k com Take(k).

Resultado: 1 aparece 3×, 2 aparece 2×, 3 aparece 1× → Top 2 = [1, 2].

public IList<int> TopKFrequent(int[] nums, int k)
{
    var counts = new Dictionary<int, int>();
    foreach (var num in nums)                         // ← fase 1: conta frequências
    {
        if (!counts.ContainsKey(num))
            counts.Add(num, 0);
        counts[num]++;
    }

    return counts
        .OrderByDescending(pair => pair.Value)         // ← fase 2: ordena por freq.
        .Select(pair => pair.Key)
        .Take(k)                                        // ← pega top k
        .ToList();
}
