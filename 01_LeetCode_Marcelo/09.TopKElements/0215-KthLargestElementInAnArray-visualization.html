0215 - Kth Largest Element in an Array

Quick 3-way Partition Sort parcial — padrão Top K Elements

Como funciona?

Input de teste: nums = [3, 2, 1, 5, 6, 4], k = 2 → resultado esperado: 5

O algoritmo usa Quick 3-way Sort parcial (descending). A cada partição, os elementos são divididos em 3 zonas em relação ao pivot:

• > pivot (à esquerda)  ·  = pivot (centro)  ·  < pivot (à direita)

A chave da eficiência: se o índice alvo k-1 já está numa zona igual ao pivot, paramos. Se está à esquerda ou direita, recursamos apenas nessa metade (as outras são descartadas com lo ≥ k ou hi < k-1).

Complexidade: Tempo O(n) esperado · Espaço O(log n) pilha

private void Quick3WaySort(int[] nums, int lo, int hi, int k)
{
    if (lo >= hi)    return;                         // base: range unitário
    if (lo >= k)     return;                         // ← poda: alvo já à esquerda
    if (hi < k - 1)  return;                         // ← poda: alvo já à direita

    // particiona em 3 zonas (descending): [>pivot] [=pivot] [<pivot]
    int lt = hi, gt = lo, i = lo, pivot = nums[i];
    while (i <= lt) {
        if      (nums[i] > pivot) Swap(nums, gt++, i);    // vai p/ esquerda
        else if (nums[i] < pivot) Swap(nums, lt--, i);    // vai p/ direita
        else                      i++;                      // igual: avança
    }
    Quick3WaySort(nums, lo, gt - 1, k);                   // ← recurse esquerda
    Quick3WaySort(nums, lt + 1, hi, k);                   // ← recurse direita
}
// return nums[k - 1]  →  resultado final
