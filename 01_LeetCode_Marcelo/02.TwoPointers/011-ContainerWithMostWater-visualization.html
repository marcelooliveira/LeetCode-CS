🫙 Container With Most Water

Two Pointers — LeetCode Problema 011

📚 Qual e o Problema?

Dado um array height onde cada elemento representa a altura de uma parede, encontre as duas paredes que formam o container que armazena mais agua. A largura do container e a distancia entre as duas paredes; a altura e limitada pela parede mais curta.

Este visualizador usa o teste MaxAreaTest_General_2:
height = [1, 2, 3, 4, 5, 6, 7, 8, 9]  →  Resultado esperado: 20  (paredes de altura 4 e 9, largura 5)

A solucao usa Two Pointers: left começa na extremidade esquerda e right na direita. A cada passo move-se o ponteiro da parede mais baixa para dentro, pois mover a mais alta so poderia diminuir a area.

🔍 Passos do Algoritmo

1 Inicializacao: left = 0, right = n-1, result = 0
2 Calcular area: area = min(h[left], h[right]) × (right - left)  |  result = max(result, area)
3 Mover ponteiro esquerdo: h[left] ≤ h[right] → avanca left ate encontrar uma parede mais alta que a atual
4 Mover ponteiro direito: h[left] > h[right] → recua right ate encontrar uma parede mais alta que a atual
5 ✅ Resultado: retornar result quando left ≥ right

⏱️ Complexidade de Tempo

O(n) — Linear

Cada elemento e visitado no maximo uma vez: left so avanca, right so recua.

💾 Complexidade de Espaco

O(1) — Constante

Apenas tres variaveis: left, right e result.

💻 Implementacao em C#

public int MaxArea(int[] height)
{
    int left = 0, right = height.Length - 1;
    int result = 0;

    while (left < right)
    {
        var area = Math.Min(height[left], height[right]) * (right - left);
        result = Math.Max(result, area);

        if (height[left] <= height[right])
        {
            var temp = height[left];
            do left++;
            while (left < right && height[left] <= temp); // skip walls that can't improve
        }
        else
        {
            var temp = height[right];
            do right--;
            while (left < right && height[right] <= temp);
        }
    }
    return result;
}
