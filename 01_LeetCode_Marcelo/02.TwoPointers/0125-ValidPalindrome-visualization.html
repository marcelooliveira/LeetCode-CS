📚 Qual e o Problema?

Dado uma string s, determine se ela e um palindromo, considerando apenas caracteres alfanumericos e ignorando maiusculas/minusculas.

Este visualizador usa o teste IsPalindromeTest_1:
Input: "A man, a plan, a canal: Panama"  →  Resultado esperado: true

A solucao usa Two Pointers: head comeca na esquerda e tail na direita. Caracteres nao-alfanumericos sao pulados em ambos os lados. Se s[head] != s[tail] (ignorando case), retorna false imediatamente.

🔍 Passos do Algoritmo

1 Inicializacao: converter para lowercase, head = 0, tail = s.Length - 1
2 Pular nao-alfanumerico em head: s[head] nao e letra nem digito → head++
3 Pular nao-alfanumerico em tail: s[tail] nao e letra nem digito → tail--
4 Comparar: s[head] == s[tail]? → head++, tail--
5 Mismatch! s[head] != s[tail] → retorna false
6 ✅ Resultado: head > tail → todos os pares verificados, retorna true

⏱️ Complexidade de Tempo

O(n) — Linear

Cada caractere e visitado no maximo uma vez: head so avanca, tail so recua.

💾 Complexidade de Espaco

O(n) — String lowercase

Cria uma copia lowercase da string. Poderia ser O(1) com comparacao case-insensitive direta.

💻 Implementacao em C#

public bool IsPalindrome(string s)
{
    if (string.IsNullOrWhiteSpace(s)) return true;
    s = s.ToLower();

    int head = 0, tail = s.Length - 1;
    while (head <= tail)
    {
        if (!IsAlNum(s[head]))       // pular nao-alfanumerico
            head++;
        else if (!IsAlNum(s[tail]))
            tail--;
        else if (s[head] != s[tail]) // mismatch
            return false;
        else
        {
            head++; tail--;              // par verificado
        }
    }
    return true;
}

// IsAlNum equivalente: (c >= 'a' && c <= 'z') || (c >= '0' && c <= '9')
