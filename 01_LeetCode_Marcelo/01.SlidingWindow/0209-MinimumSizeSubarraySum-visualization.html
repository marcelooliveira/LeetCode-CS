📏 Minimum Size Subarray Sum

Sliding Window Dinamica — LeetCode Problema 0209

📚 Qual e o Problema?

Dado um array de inteiros positivos nums e um inteiro positivo target, retorne o menor comprimento de um subarray contíguo cuja soma seja ≥ target. Retorne 0 se nenhum existir.

Este visualizador usa o caso de teste MinSubArrayLen_1:
target = 7  |  nums = [2, 3, 1, 2, 4, 3]  →  Resultado esperado: 2  (subarray [4, 3])

A solucao usa Sliding Window Dinâmica: o ponteiro right expande a janela somando elementos; assim que sum ≥ target, o ponteiro left encolhe a janela removendo elementos pela esquerda enquanto a condicao ainda for satisfeita, registrando o menor comprimento visto.

🔍 Passos do Algoritmo (execucao atual)

1 Inicializacao: result = int.MaxValue, sum = 0, left = 0
2 Expandir: sum += nums[right] — aumenta a janela pela direita
3 Verificar: while (sum ≥ target) — a soma atingiu o alvo?
4 Registrar: result = Math.Min(result, right − left + 1) — esta janela e menor que o melhor atual?
5 Encolher: sum -= nums[left++] — remove o elemento mais a esquerda e volta ao passo 3
6 ✅ Resultado: return result == int.MaxValue ? 0 : result

⏱️ Complexidade de Tempo

O(n) — Linear

Cada elemento e adicionado pela direita exatamente uma vez e removido pela esquerda no maximo uma vez. Total: até 2n operacoes.

💾 Complexidade de Espaco

O(1) — Constante

Apenas tres variaveis extra: result, sum e left. Independente do tamanho do array.

💻 Implementacao em C#

public int MinSubArrayLen(int s, int[] nums)
{
    var result = int.MaxValue;
    var sum    = 0;
    var left   = 0;

    for (int i = 0; i < nums.Length; i++)
    {
        sum += nums[i];                      // expande: inclui o elemento direito

        while (sum >= s)                       // atingiu o alvo?
        {
            result = Math.Min(result, i - left + 1);  // registra menor comprimento
            sum -= nums[left++];             // encolhe: remove o elemento esquerdo
        }
    }

    return result == int.MaxValue ? 0 : result;
}
