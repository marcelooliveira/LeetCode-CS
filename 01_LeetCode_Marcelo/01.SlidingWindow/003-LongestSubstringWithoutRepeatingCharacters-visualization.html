🔤 Longest Substring Without Repeating Characters

Sliding Window com Mapa de Posicoes — LeetCode Problema 003

📚 Qual e o Problema?

Dado uma string s, encontre o comprimento da maior substring sem caracteres repetidos.

Este visualizador usa o caso de teste LengthOfLongestSubstringTest_LongString:
Input: "abcdefghijklmnopqrstuvwxyzabcdefghijklmnopqrstuvwxyz" (52 chars)  →  Resultado esperado: 26

A solucao usa Sliding Window com um dicionario que rastreia a ultima posicao de cada caractere. A janela [lastRepeatPos+1, i] garante que nunca existam duplicatas dentro dela. Quando um caractere repetido e encontrado dentro da janela, o limite esquerdo avanca automaticamente para apos sua ultima ocorrencia.

🗺️ Mapa de Posicoes — Dictionary<int, int> map

Cada celula mostra a ultima posicao registrada do caractere no mapa. ■ Verde = esta dentro da janela atual  |  ■ Amarelo = acabou de ser atualizado  |  ■ Cinza = esta fora da janela (posicao antiga)

🔍 Passos do Algoritmo (execucao atual)

1 Inicializacao: map = {}, maxLen = 0, lastRepeatPos = -1
2 Loop: para cada i de 0 ate s.Length - 1 — percorrer a string
3 ⚠️ Repeticao detectada: map.ContainsKey(s[i]) && lastRepeatPos < map[s[i]]
→ lastRepeatPos = map[s[i]] — o limite esquerdo salta para apos a duplicata
4 Atualizar maximo: se maxLen < i − lastRepeatPos
→ maxLen = i − lastRepeatPos — nova janela maxima!
5 Registrar posicao: map[s[i]] = i — guarda ou atualiza posicao do char
6 ✅ Resultado: retornar maxLen

⏱️ Complexidade de Tempo

O(n) — Linear

Uma unica passagem pela string. O ponteiro direito avanca exatamente n vezes.

💾 Complexidade de Espaco

O(k) — onde k = tamanho do alfabeto

O dicionario armazena no maximo k entradas (26 para strings exclusivamente alfabeticas).

💻 Implementacao em C#

public int LengthOfLongestSubstring(string s)
{
    if (string.IsNullOrEmpty(s)) return 0;

    var map = new Dictionary<int, int>();
    var maxLen = 0;
    var lastRepeatPos = -1;

    for (int i = 0; i < s.Length; i++)
    {
        // Se o char esta no mapa E sua posicao esta dentro da janela atual:
        if (map.ContainsKey(s[i]) && lastRepeatPos < map[s[i]])
            lastRepeatPos = map[s[i]];   // empurra limite esquerdo

        // A janela atual tem tamanho (i - lastRepeatPos); e a maior ate agora?
        if (maxLen < i - lastRepeatPos)
            maxLen = i - lastRepeatPos;

        // Registra/atualiza a posicao do char no mapa:
        map[s[i]] = i;
    }

    return maxLen;
}
