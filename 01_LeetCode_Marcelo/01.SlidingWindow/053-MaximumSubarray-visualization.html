🎯 Maximum Subarray Problem

Algoritmo de Kadane - Visualizacao Interativa

📚 O que e o Problema do Subarray Maximo?

Dado um array de numeros inteiros, o objetivo e encontrar o subarray contiguo (sequencia de elementos adjacentes) que possui a maior soma possivel.

Por exemplo, no array [-2, 1, -3, 4, -1, 2, 1, -5, 4], o subarray [4, -1, 2, 1] tem a maior soma = 6.

O Algoritmo de Kadane resolve este problema de forma eficiente em O(n) tempo e O(1) espaco, fazendo apenas uma passagem pelo array!

🔍 Passos do Algoritmo

1 Inicializacao: currentSum = nums[0], maxSum = nums[0]
2 Para cada elemento (i = 1 ate n-1): Percorrer o array
3 Decisao: currentSum = Math.Max(nums[i] + currentSum, nums[i])
"Devo continuar o subarray anterior ou comecar um novo?"
4 Atualizacao: maxSum = Math.Max(maxSum, currentSum)
"Esta e a melhor soma encontrada ate agora?"
5 Resultado: Retornar maxSum

⏱️ Complexidade de Tempo

O(n) - Linear

Fazemos apenas uma passagem pelo array, visitando cada elemento uma unica vez.

💾 Complexidade de Espaco

O(1) - Constante

Usamos apenas duas variaveis (currentSum e maxSum), independente do tamanho do array.

💻 Implementacao em C#

public int MaxSubArray(int[] nums)
{
	if (nums.Length == 0)
		return int.MinValue;

	var currentSum = nums[0];
	var maxSum = nums[0];

	for (int i = 1; i < nums.Length; i++)
	{
		// Decide se continua o subarray ou comeca um novo
		currentSum = Math.Max(nums[i] + currentSum, nums[i]);

		// Atualiza a soma maxima se necessario
		maxSum = Math.Max(maxSum, currentSum);
	}

	return maxSum;
}
