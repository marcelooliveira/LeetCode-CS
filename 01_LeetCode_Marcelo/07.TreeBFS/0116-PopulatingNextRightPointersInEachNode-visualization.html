0116 - Populating Next Right Pointers In Each Node

Algoritmo iterativo nível a nível sem espaço extra — padrão Tree BFS

Como funciona?

Input de teste: árvore binária perfeita [1, 2, 3, 4, 5, 6, 7]

O algoritmo percorre a árvore nível a nível usando o ponteiro next já existente. Para cada nível, itera pelos nós do nível via cur = levelStart:

• cur.left.next = cur.right (conecta filhos do mesmo pai)

• cur.right.next = cur.next.left (conecta filho direito ao filho esquerdo do próximo pai)

Resultado esperado: 1→#, 2→3→#, 4→5→6→7→#

Complexidade: Tempo O(n) · Espaço O(1)

Velocidade: 900ms

public TreeLinkNode Connect(TreeLinkNode root)
{
    TreeLinkNode levelStart = root;
    while (levelStart != null)            // ← itera níveis
    {
        TreeLinkNode cur = levelStart;
        while (cur != null)                // ← itera nós do nível via next
        {
            if (cur.left != null)
                cur.left.next = cur.right;          // ← filhos do mesmo pai
            if (cur.right != null && cur.next != null)
                cur.right.next = cur.next.left;     // ← cruzamento entre pais
            cur = cur.next;
        }
        levelStart = levelStart.left;               // ← desce nível
    }
    return root;
}
