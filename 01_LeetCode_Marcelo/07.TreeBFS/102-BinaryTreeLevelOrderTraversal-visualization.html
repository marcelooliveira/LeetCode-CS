102 - Binary Tree Level Order Traversal

Algoritmo BFS (Breadth-First Search) com Queue — padrão Tree BFS

Como funciona?

Dado o input de teste: [3, 9, 20, null, null, 15, 7]

A árvore tem raiz 3, filho esquerdo 9 e filho direito 20. O nó 20 tem filhos 15 e 7.

O algoritmo usa uma Queue (fila). A cada iteração do laço externo, processa todos os nós do nível atual (tamanho da fila), adicionando os filhos para o próximo nível. O resultado esperado é: [[3], [9, 20], [15, 7]].

public IList<IList<int>> LevelOrder(TreeNode root)
{
    var results = new List<IList<int>>();
    if (root == null) return results;
    queue.Enqueue(root);                         // ← enfileira raiz

    while(queue.Count > 0)                     // ← laço de nível
    {
        var result = new List<int>();
        var levelSize = queue.Count;             // ← tamanho do nível
        for (int i = 0; i < levelSize; i++)
        {
            var node = queue.Dequeue();           // ← processa nó
            result.Add(node.val);
            if (node.left != null)              // ← enfileira filhos
                queue.Enqueue(node.left);
            if (node.right != null)
                queue.Enqueue(node.right);
        }
        results.Add(result);                     // ← salva nível
    }
    return results;
}
