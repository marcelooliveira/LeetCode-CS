0543 - Diameter of Binary Tree

DFS pós-ordem com variável global answer — padrão Tree DFS

Como funciona?

Input de teste: [1, 2, 3, 4, 5] → resultado esperado: 3

O algoritmo usa DFS pós-ordem: processa os filhos antes do pai. A função GetEdgeDepth(node) retorna a profundidade em arestas do nó. Para cada nó, o diâmetro que passa por ele é left + right. A variável global answer guarda o máximo encontrado.

No nó 2: left=1 (de 4), right=1 (de 5) → answer = max(0, 1+1) = 2. No nó 1: left=2 (de 2), right=1 (de 3) → answer = max(2, 2+1) = 3.

Complexidade: Tempo O(n) · Espaço O(h) onde h é a altura da árvore

private int answer;

public int DiameterOfBinaryTree(TreeNode root)
{
    GetEdgeDepth(root);
    return answer;
}

private int GetEdgeDepth(TreeNode root)
{
    if (root == null) return 0;                           // base case

    var left  = GetEdgeDepth(root.left);                   // ← pós-ordem: recurse esquerda
    var right = GetEdgeDepth(root.right);                  // ← pós-ordem: recurse direita

    this.answer = Math.Max(this.answer, left + right);     // ← atualiza diâmetro global
    return Math.Max(left, right) + 1;                      // ← retorna profundidade do nó
}
