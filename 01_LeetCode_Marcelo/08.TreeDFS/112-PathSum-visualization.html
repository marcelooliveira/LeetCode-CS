Como funciona?

Input de teste: árvore [5,4,8,11,null,13,4,7,2,null,null,null,1], target = 22

O algoritmo percorre a árvore com DFS, subtraindo o valor de cada nó da soma restante. Ao chegar em uma folha, verifica se a soma restante é 0 (soma exata encontrada). Se sim, retorna true; caso contrário, faz backtrack.

Caminho encontrado: 5 → 4 → 11 → 2 = 5+4+11+2 = 22 ✅

Complexidade: Tempo O(n) · Espaço O(h) onde h é a altura da árvore

public bool HasPathSum(TreeNode root, int sum)
{
    if (root == null) return false;             // nó nulo → false

    sum = sum - root.val;                               // ← subtrai valor do nó

    if (root.left == null && root.right == null)
        return sum == 0;                            // ← folha: soma restante deve ser 0

    return HasPathSum(root.left,  sum)
        || HasPathSum(root.right, sum);                 // ← tenta esquerda ou direita
}
