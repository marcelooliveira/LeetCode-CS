📖 Qual e o Problema?

Dado um inteiro n, substitua-o pela soma dos quadrados de seus digitos. Repita ate n == 1 (feliz!) ou ate ver um numero repetido (ciclo = infeliz).

🔍 Passos do Algoritmo

1 Loop while: enquanto n nao estiver em map, continuar. Se estiver, ha ciclo → retorna false.
2 Adicionar ao map: map.Add(n). Guardar n para detectar ciclo futuro.
3 Calcular novo valor: extrair cada digito d = n % 10, acumular d*d, depois n /= 10.
4 Checar == 1: se newValue == 1 → numero feliz! Retorna true imediatamente.
5 Continuar: n = newValue e voltar ao loop. Repetir ate encontrar 1 (feliz) ou ciclo (infeliz).

⏱️ Complexidade de Tempo

O(log n) por iteracao

O numero de digitos de n e O(log n). A sequencia converge para 1 ou entra em ciclo em O(log n) iteracoes.

💾 Complexidade de Espaco

O(log n)

O HashSet armazena os numeros vistos. O numero de valores distintos na sequencia antes do ciclo e O(log n).

💻 Implementacao em C#

public bool IsHappy(int n)
{
    var map = new HashSet<int>();

    while (!map.Contains(n))
    {
        map.Add(n);
        var newValue = 0;
        while (n > 0)
        {
            var digit = n % 10;        // ultimo digito
            newValue += digit * digit;   // acumula digito²
            n /= 10;                  // remove ultimo digito
        }
        if (newValue == 1) return true;   // feliz!
        n = newValue;
    }
    return false;  // ciclo detectado
}
