🔍 Find The Duplicate Number

Floyd's Cycle Detection em Array como Grafo Funcional — LeetCode 0287

📖 Qual e o Problema?

Dado um array de n+1 inteiros no intervalo [1,n], encontrar o numero duplicado. Sem modificar o array e usando O(1) de espaco extra.

Este visualizador usa FindDuplicateTest_1:
nums = [1, 3, 4, 2, 2] → resultado esperado: 2

O array e tratado como um grafo funcional: indice i aponta para nums[i]. Como ha duplicata, ha um ciclo. O algoritmo de Floyd detecta o ciclo (Fase 1) e encontra a entrada do ciclo = duplicata (Fase 2).

Grafo: 0→1→3→[2→4→2]→... (ciclo: 2↔4, entrada em 2 = duplicata)

🔍 Passos do Algoritmo

1 Fase 1 - Init: tortoise = hare = nums[0]. Ambos partem da mesma posicao inicial.
2 Fase 1 - Avancar: tortoise = nums[tortoise] (+1 salto). hare = nums[nums[hare]] (+2 saltos). Repetir ate tortoise==hare.
3 Fase 1 - Encontro: tortoise==hare no interior do ciclo. O ponto de encontro esta dentro do ciclo.
4 Fase 2 - Init: ptr1 = nums[0], ptr2 = tortoise (ponto de encontro). Reiniciar ptr1 do inicio.
5 Fase 2 - Avancar: ptr1 = nums[ptr1], ptr2 = nums[ptr2]. Ambos +1 salto. Quando se encontrarem, esse indice = duplicata.
6 Resultado: ptr1==ptr2 = entrada do ciclo = valor duplicado. Retornar ptr1.

⏱️ Complexidade de Tempo

O(n)

Fase 1 converge em O(λ+μ) passos. Fase 2 converge em mais μ passos. Total linear.

💾 Complexidade de Espaco

O(1)

Apenas 4 ponteiros inteiros extras. Nao modifica o array de entrada.

💻 Implementacao em C#

public int FindDuplicate(int[] nums)
{
    // Fase 1: encontrar ponto de encontro no ciclo
    int tortoise = nums[0], hare = nums[0];
    do
    {
        tortoise = nums[tortoise];           // +1 salto
        hare     = nums[nums[hare]];         // +2 saltos
    } while (tortoise != hare);

    // Fase 2: encontrar entrada do ciclo (= duplicata)
    int ptr1 = nums[0], ptr2 = tortoise;
    while (ptr1 != ptr2)
    {
        ptr1 = nums[ptr1];                   // +1 salto
        ptr2 = nums[ptr2];                   // +1 salto
    }
    return ptr1;                             // entrada do ciclo = duplicata
}
