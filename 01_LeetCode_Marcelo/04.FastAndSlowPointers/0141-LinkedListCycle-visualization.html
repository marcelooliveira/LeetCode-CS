🔗 Linked List Cycle

Ponteiros Lento e Rapido (Floyd's Tortoise & Hare) — LeetCode 0141

📖 Qual e o Problema?

Dada a cabeca de uma lista ligada, determinar se ela contem um ciclo. Um ciclo existe quando algum no aponta de volta para um no anterior.

🔍 Passos do Algoritmo

1 Verificacao inicial: se head == null || head.next == null → retorna false imediatamente.
2 Inicializar: slow = head, fast = head.next. Entrar em while(slow != fast).
3 Checar terminacao: se fast == null || fast.next == null → lista finita, sem ciclo → retorna false.
4 Avancar: slow = slow.next (+1), fast = fast.next.next (+2). fast se aproxima do slow pelo ciclo.
5 Encontro! slow == fast → eles se encontraram dentro do ciclo → retorna true.

⏱️ Complexidade de Tempo

O(n)

Se ha ciclo de comprimento λ e cauda de comprimento μ, o encontro ocorre em no maximo μ + λ passos.

💾 Complexidade de Espaco

O(1)

Apenas dois ponteiros extras (slow e fast) — constante, independente do tamanho da lista.

💻 Implementacao em C#

public bool HasCycle(ListNode head)
{
    if (head == null || head.next == null) return false;

    ListNode slow = head, fast = head.next;
    while (slow != fast)
    {
        if (fast == null || fast.next == null) return false;
        slow = slow.next;
        fast = fast.next.next;   // fast avanca 2x mais rapido
    }
    return true;              // slow == fast: encontro no ciclo
}
